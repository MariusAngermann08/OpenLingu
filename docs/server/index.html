Server Overview

The OpenLingu server handles user authentication, content delivery, and data storage. It provides a RESTful API that the client application uses to interact with the system.

Server Administration

Starting the Server

To start the OpenLingu server:

python -m server.main

Configuration

Server configuration is stored in server/config.ini. You can modify settings such as:

  • Server port and host
  • Database connection
  • Session settings
  • File storage locations

User Management

Standard users can register through the client interface. To create a contributor account, use the admin tools while the server is offline:

python -m server.admin.tools --add-contributor username

Backup and Restore

Regularly back up your server data. The server stores data in the following locations:

  • Database: server/data/database.db
  • Uploaded files: server/data/uploads/
  • Lection content: server/data/lections/

Troubleshooting

Server won't start

Check the following:

  • Required ports are not in use
  • Database file is not corrupted
  • File permissions are set correctly

Users can't connect

Verify:

  • Server is running and accessible
  • Firewall allows traffic on the configured port
  • Client is using the correct server URL
