Creating Lections

The Lection Creator is a powerful tool for designing interactive language learning content. You must have contributor privileges to use these features.

Getting Started

  1. Log in with a contributor account
  2. Navigate to the Lection Creator from the main menu
  3. Create a new lection or open an existing one

Lection Structure

Each lection consists of:

  • Metadata: Title, description, language, difficulty level
  • Pages: Individual screens in the lection
  • Widgets: Interactive elements on each page

Available Widgets

Text

Display formatted text with support for Markdown

Image

Add images with optional captions

Audio

Embed audio files with playback controls

Multiple Choice

Create questions with single or multiple correct answers

Matching Pairs

Create matching exercises with draggable elements

Fill in the Blanks

Create cloze tests with text input fields

Best Practices

  • Keep lections focused on a specific topic or grammar point
  • Use a variety of widget types to maintain engagement
  • Include clear instructions for each interactive element
  • Test your lection thoroughly before publishing
  • Use images and audio to support learning

Publishing Lections

When your lection is ready:

  1. Click "Save" to save your progress
  2. Use "Preview" to test the lection
  3. Click "Publish" to make it available to users
  4. You can update published lections at any time
