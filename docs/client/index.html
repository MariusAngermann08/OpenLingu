Using the OpenLingu Client

The OpenLingu client provides a user-friendly interface for language learners to access and complete lections.

Connecting to a Server

When you first launch the client, you'll be prompted to connect to a server. You can:

  • Use the default local server (http://localhost:8000)
  • Connect to a different server by entering its URL
  • Switch servers later in the settings menu

User Accounts

To use OpenLingu, you'll need to create a user account:

  1. Click "Create Account" on the login screen
  2. Enter your desired username and password
  3. Complete the registration form
  4. Log in with your new credentials

Browsing Lections

Once logged in, you can:

  • Browse available languages
  • View lections within each language
  • See your progress on each lection
  • Filter lections by difficulty or completion status

Completing Lections

Lections consist of multiple pages with different interactive widgets:

  • Multiple Choice: Select the correct answer
  • Matching: Connect related items
  • Fill in the Blank: Type the missing words
  • Drag and Drop: Arrange items in the correct order

Offline Mode

To use lections offline:

  1. While online, navigate to the lection you want to download
  2. Click the download icon
  3. Once downloaded, you can access the lection offline
  4. Your progress will sync the next time you're online
